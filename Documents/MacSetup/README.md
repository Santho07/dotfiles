# Mac Setup

There is two scripts to setup new mac.

1. `osx.sh` : Apply Mac Tweaks

2. `brew.sh` : Install Useful apps and libraries

Run them in order on a new system..

```
  chmod +x osx.sh
  chmod +x brew.sh

  ./osx.sh
  ./brew.sh
```
