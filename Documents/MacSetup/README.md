# Mac Setup

There is two scripts to setup new mac. 

1. osx.sh
  
  This repository contains useful tweaks for mac

2. brew.sh
   
   To install useful softwares.


Run them in order..

   ```
     ./osx.sh
     ./brew.sh
   ```
